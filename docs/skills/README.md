# Development Skills for the Learning Coach Project

This folder contains implementation guidance and execution checklists for building the AI-powered learning and interview coach.

## Skill Files
- foundation.md: project setup, architecture, local environment, and repository structure
- rag-and-question-bank.md: ingestion, embedding, vector DB, and question bank workflows
- agent-orchestration.md: agent design, orchestration, prompts, and workflow routing
- mcp-integration.md: secure enterprise integrations and tool access policy
- ux-and-delivery.md: frontend flows, trainer workflows, deployment, and launch readiness
