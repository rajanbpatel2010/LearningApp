# Foundation Skill

## Objective
Set up the project foundation so development can begin in a repeatable way.

## Deliverables
- Backend starter app
- Frontend starter app
- Database schema and migrations
- Docker-based local environment
- CI pipeline

## Implementation Checklist
1. Create a FastAPI backend with health endpoints.
2. Create a React or Angular frontend shell.
3. Configure PostgreSQL and Redis locally.
4. Define users, sessions, questions, reports, and documents models.
5. Add environment config and secrets handling.
6. Add logging, health checks, and basic tests.
7. Create a installer setup for local development or make it ready to deploy on vercel.

## Definition of Ready
- The app can run locally with a single command.
- Developers can create and apply database migrations.
- The repo contains a baseline CI workflow.
